JobMate: Your AI Career Assistant

JobMate is an open-source AI-powered web app built using Streamlit and Hugging Face Transformers. It helps job seekers craft professional resume summaries, cover letters, prepare for mock interviews, and extract key skills from job descriptions.

Features

- Resume Summary Generator
- Cover Letter Writer
- Mock Interview Q&A
- Job Description Skill Extractor

Built With

- Streamlit
- Hugging Face Transformers (flan-t5-base)
- PyTorch
- PIL (for logo support)

How to Run Locally

```bash
pip install -r requirements.txt
streamlit run JobMate_app.py
